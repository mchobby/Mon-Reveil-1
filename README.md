Le boitier contient 5 parties (disponible dans le dossier 3D):
- reveil-support.stl 
- reveil-couvercle.stl 
- reveil-frontal.stl 
- reveil-hack.stl
- reveil-petites-pieces.stl

Pour plus d'information vistez le [wiki](https://wiki.mchobby.be/index.php?title=Mon-Reveil-3D)
